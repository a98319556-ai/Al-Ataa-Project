# منصة العطاء الموحدة - v2.0

## التغييرات في هذا الإصدار
- **لوجين موحد**: صفحة دخول واحدة `/login` تحدد تلقائياً أدمن أو متبرع
- **الصفحة الرئيسية الديناميكية**: تعرض الجمعيات الشريكة والحملات الحقيقية من الداتابيز
- **إضافة جمعية وحملة**: فورمات نظيفة وتعمل بشكل صحيح
- **حذف قديم**: حُذفت صفحة الجمعيات من قسم المتبرعين كما طلبت

## المسارات الرئيسية
| الصفحة | المسار |
|--------|--------|
| الرئيسية (مع الجمعيات والحملات) | `/` |
| **لوجين موحد** | `/login` |
| إنشاء حساب متبرع | `/signup` |
| تسجيل الخروج | `/logout` |
| لوحة المتبرع | `/donor/home` |
| لوحة الأدمن | `/admin` |
| المخزن | `/admin/inventory` |
| الجمعيات | `/admin/charities` |

## كيف يعمل اللوجين الموحد
1. تدخل اسم المستخدم وكلمة المرور
2. النظام يبحث في جدول `admins` أولاً
3. إذا وجده → يروح للوحة الأدمن
4. إذا لم يجده → يبحث في جدول `users` (المتبرعين)
5. إذا وجده → يروح لصفحة المتبرع
6. إذا لم يجده → رسالة خطأ

## بيانات الدخول التجريبية

### المشرفون (جدول admins)
| المستخدم | كلمة المرور | الصفحة |
|----------|------------|--------|
| admin | admin123 | /admin |
| manager | manager123 | /admin |

### المتبرعون (جدول users)
سجّل حساب جديد من `/signup`

## الإعداد
```bash
# 1. استيراد قاعدة البيانات
mysql -u root -p < database.sql

# 2. تثبيت المتطلبات
pip install -r requirements.txt

# 3. تشغيل التطبيق
python app.py
```
